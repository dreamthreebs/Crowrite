# 1122-some markdown format

### Center the things

```markdown
<div align="center">

| 表头1  | 表头2|
| ---------- | -----------|
| 表格单元   | 表格单元   |
| 表格单元   | 表格单元   |

</div>
```

| 表头1  | 表头2  |
| ---- | ---- |
| 表格单元 | 表格单元 |
| 表格单元 | 表格单元 |

### Nested title

```markdown
<details>
<summary>Linux环境</summary>

##### 编译
xxxx

##### 安装
xxxx
</details>
```

<details>

<summary>Linux环境</summary>

**编译**

xxxx

**安装**

xxxx

</details>

### TO-DO LISTS:

```markdown
- [ ] Task 1
- [ ] Task 2
- [x] Completed Task
```

* [ ] Task 1
* [ ] Task 2
* [x] Completed Task
